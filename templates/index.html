About

RocketDB is an open-source database aiming to collect comprehensive information about rockets and their launches. You can help the project in the following ways:

  • Add new data to the database.
  • Update information on existing rockets and launches.
  • Report errors or inconsistencies in the data.
  • Share the project with others interested in rockets.
  • Contribute to the project code or documentation on GitHub.
